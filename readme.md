### Development

Projet en cours de développement, a ne pas se servir en production ou je ne sais quoi xD 

Node:
```sh
$ npm install
$ bower install
$ gulp
```

Composer:
```sh
$ composer install
$ composer update
```



Testing :
```sh
$ .\vendor\bin\codecept run
```

```properties
COUCHDB_HOST=localhost:5984
COUCHDB_DATABASE=database
```
